För att starta programmet öppna filen som heter "STARTA PROGRAMMET.BAT" och skriv in matchId som går hitta på https://stats.swehockey.se/

Exempelvis: "629537", från: https://stats.swehockey.se/Game/Events/629537

Därefter kan man låta det vara. Det uppdateras varje 30 sekunder.

I OBS ska man använda sig av en Bild som källa och sätt källan till "modified-image.png"
